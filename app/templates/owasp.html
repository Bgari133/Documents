OWASP Top 10:2021 – Mapping to VulnLab

This page maps each OWASP Top 10:2021 category to labs in this site. Use it to plan your testing and understand impact.

CategoryLabs in VulnLabImpact (summary)
A01:2021 Broken Access Control IDOR, Path Traversal, CSRF Access other users’ data, read arbitrary files, perform actions as victim.
A02:2021 Cryptographic Failures Crypto, Config (plaintext/keys) Credential theft, session forgery, PII disclosure.
A03:2021 Injection SQLi, Command Injection, XXE; XSS (client-side) Data breach, RCE, account takeover, file read.
A04:2021 Insecure Design Design flaws across labs (e.g. trust user input, no defense in depth) Harder to fix with patches; requires design changes.
A05:2021 Security Misconfiguration Default creds, Debug mode, Config exposure Easier exploitation, credential leak, info disclosure.
A06:2021 Vulnerable and Outdated Components See README; use real scanners (e.g. Snyk, Dependabot) in real apps Known CVEs lead to RCE, XSS, or data loss.
A07:2021 Identification and Authentication Failures Broken Auth (plaintext, no rate limit) Brute force, credential stuffing, session hijack.
A08:2021 Software and Data Integrity Failures Insecure Deserialization RCE via malicious serialized payloads.
A09:2021 Security Logging and Monitoring Failures Not implemented (no logging of failed logins, etc.) Attacks go undetected; no audit trail.
A10:2021 Server-Side Request Forgery (SSRF) SSRF Access internal services, cloud metadata, bypass firewall.

Cross-cutting: Reflected and Stored XSS (A03/CWE-79); Open Redirect (CWE-601, phishing); Clickjacking (CWE-1021, UI redressing).
